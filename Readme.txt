# Scenario


## Description
This project is a Clinic Management System developed using Object-Oriented Programming and SQLite. It is based on an ER diagram designed to manage patient and doctor information.

## Features
- List full details of senior patients (age > 65)
- Display total number of doctors specialising in ophthalmology

## Technologies Used
- Python
- SQLite
- Object-Oriented Programming

## Files
- db_setup.py – Database creation and sample data
- main.py – OOP implementation and queries
- clinic.db – SQLite database
- ER_Diagram.png – ER diagram screenshot

## How to Run
1. Run `db_setup.py`
2. Run `main.py`
